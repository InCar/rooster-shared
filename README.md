## 前提条件
- [JDK 1.8+](http://www.oracle.com/technetwork/java/javase/downloads/index.html)
- [Gradle 3.4+](http://gradle.org/gradle-download/)
```shell
# 执行以下命令检查环境
java -version
gradle --version
```

## 编译
```SHELL
gradle assemble
```

gradle uploadArchives
